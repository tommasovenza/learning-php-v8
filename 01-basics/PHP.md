# PHP - Overview

PHP inizialmente significava Personal Home Page. Nasce nel 1994.
Dopo viene rinominato in PHP Hypertext Processor.
Con la versione 5 di PHP del 2004 il linguaggio diventa ad oggetti.
La versione 6 non verrà mai rilasciata.
Nel 2015 esce la versione 7 e nel 2020 la versione 8.

PHP è un linguaggio di scripting open source. Non essendo compilato viene interpretato dai dei programmi parser senza la necessità di essere compilato.

PHP viene definito beginner friendly. La curva di apprendimento però è tutt’altro che banale. E’ utilizzato moltissimo in ambito web. Ha un ecosistema ricco e completo. ci sono molte soluzioni pronte all’uso come sistemi per forum, piattaforme di blogging, CRM o framework per creare API.

Ancora il 79% del web è fatto con PHP. Il 60% di questi utilizza PHP 7 ed il restante 40% PHP 5.

# Installazione

## Installazione di un ambiente di sviluppo PHP

Per utilizzarlo, va prima installato, perché non è disponibile di default sul sistema operativo.
Si può installare da linea di comando oppure utilizzare un server web.

La maggior parte delle applicazioni PHP vengono sviluppate per il web ci concentreremo su impostare un ambiente di sviluppo web per PHP. (Laragon su windows e MAMP su macOS)

## BOUNDLE

Per eseguire gli script di PHP servono queste cose. Un boundle.
Ovvero:
un sistema operativo,
un web server,
un interprete PHP
ed un database (può essere utile)

LAMP acronimo per linux, apache, mysql, PHP
MAMP acronimo per macOS, apache, mysql, PHP

Questi sono gli stack utilizzati di solito per web Application PHP. Anche se ultimamente, grazie ai container ed i microservizi sta prendendo piede Nginx al posto di Apache.
